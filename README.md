# Data Science + AWS Learning Path

Proyecto de formación práctica en Ciencia de Datos y Cloud Computing.

## Semana 1

## Proyecto:
Análisis de un dataset de ventas utilizando Python.

### Procesos realizados:
- Ingesta de datos desde CSV
- Limpieza de datos (nulos, duplicados, valores inválidos)
- Transformaciones
- Creación de variables derivadas (ingresos)
- Estadísticos básicos
- Agrupaciones
- Visualización
- Interpretación de resultados

## Tecnologías:
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Objetivo:
Construir base sólida para ciencia de datos y arquitecturas cloud data.
